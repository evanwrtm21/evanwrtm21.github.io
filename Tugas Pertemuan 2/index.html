About Me

Perkenalkan nama saya Mochamad Evan Wiratama biasa dipanggil Evan. Saya lahir di Surabaya pada tanggal 21 Juni 2001. Untuk sekarang saya tinggal di Sidoarjo karena pada awalnya orang tua saya bekerja di daerah Sidoarjo.

Untuk pendidikan saya, saya menempuh sejak kelas TK hingga saat ini berkuliah di IT Telkom Surabaya. Sejak kecil saya hingga sekarang saya mempunyai hobby yaitu bermain game hingga saat ini saya pun masih bermain game jika ada waktu luang. Sebisa mungkin saya harus bisa membagi waktu bagaimana saya bisa belajar untuk menempuh cita-cita saya.
